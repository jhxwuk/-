# Навчальний проєкт з інформатики  
## Тема: «Тенета Інтернету»

Цей проєкт створено у вигляді вебсторінки HTML для захисту індивідуального навчального проєкту з інформатики.

## Мета проєкту
Узагальнити відомості про мережу Інтернет, її історію створення, апаратні засоби, основні сервіси та значення для сучасної людини.

## Вміст проєкту
Сайт містить такі розділи:
- Мета проєкту
- Історія виникнення Інтернету
- Основні сервіси Інтернету
- Значення Інтернету для людини
- Висновки

## Файли
- `index.html` — основна вебсторінка проєкту
- `README.md` — опис проєкту та інструкція з використання

## Як відкрити проєкт
1. Завантажити файл `index.html`
2. Відкрити його у будь-якому сучасному браузері:
   - Google Chrome
   - Microsoft Edge
   - Mozilla Firefox
   - Safari

## Особливості оформлення
- сучасний дизайн
- адаптивна структура
- зручна навігація по розділах
- темна кольорова тема
- плавні ефекти при наведенні

## Для редагування
Щоб змінити ім’я учня, клас або текст:
1. Відкрити файл `index.html` у текстовому редакторі
2. Знайти потрібний фрагмент тексту
3. Зберегти зміни
4. Знову відкрити файл у браузері

## Висновок
Проєкт демонструє можливості створення простої навчальної публікації засобами HTML і CSS та розкриває тему значення Інтернету в сучасному суспільстві.
